# XV-15 螺旋桨参数化采样 + BEM 筛选 + CATIA 批量建模

本项目实现:
1. 拉丁超立方采样 (LHS)
2. CATIA 批量建模导出 STEP
3. BEM 低保真筛选 Top 200

## 设计变量与范围

- 翼根弦长: 431.8 mm, 范围 300-560 mm  
- 翼中弦长: 368.3 mm, 范围 250-480 mm  
- 翼尖弦长: 304.8 mm, 范围 210-400 mm  
- 翼根扭转角: 40.25°, 范围 30-50°  
- 翼中扭转角: 18.25°, 范围 10-25°  
- 翼尖扭转角: -3.75°, 范围 -10~5°  

默认固定参数:
- 旋翼直径: 7620 mm
- 叶片数: 3

## 文件说明

- `main_workflow.py`  
  一键运行完整流程
- `xv15_sampling.py`  
  LHS 采样 + 约束检查
- `catia_batch_modeling.py`  
  CATIA 批量参数化建模
- `bem_propeller.py`  
  简化 BEM 求解器
- `xv15_bem_screening.py`  
  BEM 批量分析与筛选

## 快速开始

### 1) 安装依赖

```bash
pip install numpy pandas matplotlib pycatia
```

### 2) 运行完整流程

```bash
python main_workflow.py
```

### 3) 只做采样

```bash
python xv15_sampling.py
```

### 4) 只做 BEM 筛选

```bash
python xv15_bem_screening.py
```

### 5) 只做 CATIA 批量建模

```bash
python catia_batch_modeling.py
```

## CATIA 模板配置

在 `main_workflow.py` 和 `catia_batch_modeling.py` 中修改:

```python
TEMPLATE_FILE = r"你的 CATIA 模板路径"
OUTPUT_DIR = r"输出目录"
```

## 说明

- BEM 求解是低保真筛选, 目的是快速筛掉效率低的方案  
- 若有真实翼型极曲线数据, 可替换 `bem_propeller.py` 中的 `AirfoilData`
